Manuais de operação

Teclado 558 via web

Armar


Para armar é nescessario que o sistema esteja "pronto para armar" que é representado por uma luz verde na lateral direita abaixo do desenho de um cadeado.

Digite uma senha válida.

Uma luz vermelha ascenderá na lateral direita abaixo do cadeado sinalizando que armou!


Desarmar


Para desarmar é só digitar uma senha válida.

Caso não funcione pressione "CANCELAR" e digite novamente.


Para Anular Setores


Digite o número da zona a ser anulada com 02 digitos + EXC + SENHA, O SISTEMA ARMARÁ JA COM A ZONA ANULADA.(isso pode ser feito sequencialmente para anular mais de uma zona:ex ,02 exc, 03 exc, 04 exc)



Cadastrar senhas


Nota: A senha master de fabrica é :151515


Digite ENTER+ SENHA MASTER +ENTER (se o codigo digitado estiver correto o teclado emitirá um bip intermitente , caso o codigo esteja incorreto o teclado emitirá um bip longo) + usuário com 3 digitos (ex:003 , o sistema permite no maximo 100 usuários) + digite a senha a ser cadastrada + (O teclado emitira um bip intermitente de confirmação .


Apagar senhas


Para deletar denhas digite : ENTER + SENHA MASTER + ENTER + USUÁRIO(03 digitos) + CANCELAR + ENTER.
