Manuais de operação

Teclado intelbras

Armar


Para armar é nescessario que o sistema esteja "pronto para armar" que é representado pela letra "P" no canto inferior esquerdo do LCD.

Digite uma senha válida.

Acenderá uma luz azul abaixo do LCD sinalizando que armou!


Desarmar


Para desarmar é só digitar uma senha válida.

Caso não funcione pressione "Apagar" e digite novamente.


Para Anular Setores


Digite "ANULAR" o número do setor(s) a ser anulado com 2 digitos e "ANULAR" novamente mais a senha com permissão para anular (senha master).
