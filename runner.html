Manuais de operação

Teclado Runner

Armar


Para armar é nescessario que o sistema esteja "pronto para armar" que é representado por uma luz verde "ready" ascesa no teclado.

Digite uma senha válida + enter.

O luz vermelha "A" ascenderá sinalizando que armou!


Desarmar


Para desarmar é só digitar uma senha válida + enter.


Para Anular Setores


Digite "BYPASS" mais o setor a ser anulado com 2 digitos,pode ser mais de um setor se nescessário:Ex "BYPASS + 01, 02, + ENTER" caso tenha anulado os setores vai acender a luz "ready" .Agora é só armar!
