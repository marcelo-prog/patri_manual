Manuais de operação

Teclado Central de Alarme DSC

Armar


Para armar é nescessario que o sistema esteja "pronto para armar" que é representado por uma luz verde.

Digite uma senha válida.

A luz vermelha ascenderá sinalizando que armou!


Desarmar


Para desarmar é só digitar uma senha válida.

Caso não funcione pressione "#" e digite novamente.


Para Anular Setores


Digite " *(asterisco) " + "1" + senha + setor com 2 digitos + " # " Agora é só armar!
