Armar


Para armar é nescessario que o sistema esteja "pronto para armar" que é representadopor uma luz verde aqui.

Digite uma senha válida.

O cadeado vermelho ascenderá sinalizando que armou!


Desarmar


Para desarmar é só digitar uma senha válida.

Caso não funcione pressione "Clear" ou "Apagar" e digite novamente.
