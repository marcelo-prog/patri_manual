Manuais de operação

Teclado 558 sistema de alarme VIA WEB

Armar


Para armar é nescessario que o sistema esteja "pronto para armar" que é representado por uma luz verde.

Digite uma senha válida.

O cadeado vermelho ascenderá sinalizando que armou!


Desarmar


Para desarmar é só digitar uma senha válida.

Caso não funcione pressione "Cancelar" e digite novamente.


Para Anular Setores


Digite o setor a ser anulado com 2 digitos + "EXC" + mais a senha, e possivel anular mais de um setor digitando os setores a serem anulados todos com 2 digitos EX. 05,06,07,10 + EXC + SENHA para armar, aonde os setores 05,06,07,10 seriam anulados nesse caso.
