Teclado 636 Alarme Paradox 728/738/748

Armar


Para armar é nescessario que o sistema esteja "pronto para armar" que é representado por uma luz verde.

Digite uma senha válida.

O cadeado vermelho ascenderá sinalizando que armou!


Desarmar


Para desarmar é só digitar uma senha válida.

Caso não funcione pressione "Clear" ou "Apagar" e digite novamente.


Para Anular Setores


Digite "BYP" ou "ANLR" mais a senha, a tecla de anulação "BYP" ou "ANLR" vai ficar piscando esperando você pressionar o(s) setor(es) que precisam ser anulados após isso pressione "ENTER" .Agora é só armar!



Cadastrar senhas


Para cadastrar novas senhas na Paradox -728


Nota: A senha master de fabrica é :474747


Digite ENTER+ SENHA MASTER (se o codigo digitado estiver correto o teclado emitirá um bip intermitente e ficara piscando o enter, caso o codigo esteja incorreto o teclado emitirá um bip longo) + usuário com 2 digitos (ex:03 , o sistema permite no maximo 48 usuários) + digite a senha a ser cadastrada + enter.


Apagar senhas


Para deletar denhas digite : ENTER+ SENHA MASTER + USUÁRIO + 2ND + ENTER.
