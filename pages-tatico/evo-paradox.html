Manuais de operação

Teclado da Central EVO-paradox

Armar


Para armar é nescessario que o sistema esteja "pronto para armar" que é representado por uma luz verde.

Digite uma senha válida.

Pressione ARM o sistema iniciara o bip do tempo de saida.

NOTA:Caso o seu sistema de alarme tenha mais de uma partição o processo de arme é diferente, pois terá que digitar (0) após o "ARM" para armar todas as partições.


Desarmar


Para desarmar é só digitar uma senha válida +DISARM.

Caso não funcione pressione "Clear" ou "Apagar" e digite novamente.

NOTA:Caso o seu sistema de alarme tenha mais de uma partição o processo de desarme é diferente, pois terá que digitar (0) após o "DISARM" para desarmar todas as partições.


Para Anular Setores


Digite uma senha valida + BYP ou EXC+ os setores com 02 digitos (EX: 01,02,03,04) após digitar os setores a serem anulados pressione BYPnovamente, o sistema mostrara as zonas anuladas no display pressione ENTER. Agora é só armar!
